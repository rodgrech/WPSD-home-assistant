# Branding Assets

These assets are prepared for the next release revision.

- `logo.svg`: scalable integration mark for GitHub/README use.
- `logo.png`: generated square logo artwork.
- `icon.png`: resized square icon suitable for compact README/HACS display.
- `hacs-icon.png`: 256px variant for HACS/custom repository presentation.

Home Assistant's official integration brand images are normally managed through
the separate Home Assistant Brands process. These files keep the repository
ready for HACS/GitHub presentation and future brand submission.
